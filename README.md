# Cleaning Supply Requester

## Setup
1. Copy `.env.sample` to `.env` and fill values.
2. `npm install`
3. `npm run dev` to start Vite.
4. `npm run supabase:start` to start local Supabase.
5. `npm run seed` to import Excel data.
